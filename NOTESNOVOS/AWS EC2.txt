------------------------------------------------------Security Groups----------------------------------------------------------------------------
NA AWS VAMOS ATE REDE DE SEGURANÇA DEPOIS ATE SECURITY GROPS
POR PADRÃO QUANDO NOS CRIA NOSSA CONTA TEM UM GRUPO DE SEGURANÇA DEFAULT 
TEM UMA ABA CHAMADA REGRA DE ENTRADA 
LIBERA TODO ACESSO A SUA REDE, TANTO PARA ENTRADA QUANTO PARA SAIDA SO POR MOTIVOS DE TESTE
E POR PADRÃO TBM TEM OUTRO SECURITY GROUPS QUE SE CHAMA LAUNCH WIZZARD -1
ESSE POUCO DIFERENTE SO TEM ACESSO DE ENTRA DA PARA SSH E DE QUALQUER ORIGEM 

NOS PODEMOS MODIFICAR AO INVES DE ENTRADA GERAL NOS PODE DEIXAR SO PARA ENTRA DO NOSSO COMPUTADOR 
ENTRADA GERAL POR PADRÃO 00000
PARA DEIXA ACESSO SOMENTE NOSSO COMPUTADOR SO FAZER NO AWS NO EDITAR REGRAS DE ENTRADA TIRA O 00000 E 
COLOCAR NA PESQUISA SOMENTE MEU IP COM ISSO AWS VAI INDETIFICAR MEU IP PUBLICO SO DEIXAR MEU COMPUTADOR ACESSAR
E PODE ADD  UM DISCRIÇÃO E APERTA SALVAR 


VOCE PODE IR NO GRUPO DE SEGURAÇA E CRIAR UM
PARA LIBERA POR EXEMPLO PARA SO PARA WEB ETC
COMO VC COLOCA PORTAS QUE SE PODE ACESSAR   80 A 443
E COLOCA PARA CONFIGURA A HTTP E HTTPS E ADD UMA REGRA E ADD O IP QUE PODE ACESSA 
TANTO IPV4 QUANTO IPV6 OU OS DOIS JUNTO COLOCANDO 0.0.0.0/0::/0
MESMA COISA COM HTTPS

-----------------------------------------------IP DEDICADO---------------------------------------------------------------------

COMO CRIAR O IP DEDICADO,POR QUE PRECISO POR CAUSA QUE VC PODE TER VARIAS MAQUINAS CONTANDO COM ESSE IP, MESMO QUE VC NUNCA
DESLIQUE ESSA MAQUINA UMA HORA PODE DAR PANE VC TER QUE MUDAR TODOS IPS VAI DAR ERRO

COMO SE FAZ UM IP DEDICADO?
VAMOS EM REDES E SEGURANÇA E EM IPS ELASTICOS, E APERTAR EM ALOCAR IP E APERTA EM ALOCAR IP

E COM IP ALOCADO, VC VAI APERTA EM IP E ASSOSIAR ENDEREÇO DE IP, E FALAR QUE ESSE IP ESTA APONTANDO PARA ALGUMA INSTANCIA
OU ALGUM  LUGAR DA REDE, 
E PARA QUAL INSTANCIA SERIA ELA E APERTA EM ASSOCIAR

VAMOS APERTA EM CRIAR LOAD BALANCING, LA TEM ALGUNS TIPOS,

DAR NOME PARA ELE, E CONFIGURAÇÃO DELE 
E UMA COISA VC TEM ATENTAR E QUAL VPC TA,E MARCA SUB REDES QUE TEM REDIRECIONAR 
E SECURITY GROUPS, O DAFAULT TEM QUE ESTA HABILITADO POR QUE ?
